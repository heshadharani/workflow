USD Workflow

Universal Scene Description (USD) is an open-source interchange format that offers a unified language for transferring vast amounts of data between various 3D applications. You must enable the USD Importer Plugin to work with USD files in the Unreal Editor. Many DCC applications, such as Maya, Blender, and Cinema 4D, offer support for USD, making it a versatile option for cross-application collaboration.

Keywords: USD Importer


Pain Points

  • Occasionally, certain skeletons structures do not work with USD due to internal optimisation issues.
  • USD integration to the Unreal engine needs more documentation.
  • Even though USD is designed to handle large-scale data, it can have an impact on performance when importing complex animated scenes or characters with high-resolution meshes and textures.
  • There is a learning curve associated with understanding the USD format and features.


Best Practics

  • Ensure that DCC software supports the USD file format.
  • Maintain consistent naming conventions and organisation in the scene hierarchy.
  • Optimise your DCC applications scene before exporting to reduce impact performance.
  • Set the correct frame range, frame rate and export settings when exporting.
  • Use appropriate export settings, such as "ASCII" or "Binary" formats, based on your DCC software.
  • Apply materials to the imported USD geometry in Unreal Engine.
