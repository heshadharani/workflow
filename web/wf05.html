Live Link

Live link is an interface that allows data streaming into Unreal Engine from DCC applications. This provides the ability to edit animation in DCC applications while getting a preview of the animation inside Unreal Engine. You must enable the live link plugin to work with a live link in Unreal Engine. The live link plugin is currently available for Maya, Motionbuilder and Blender.

Keywords: Live Link, Stream


Pain Points

  • There could be latency between the DCC application and the Unreal Engine, making previewing the animation in real-time challenging.
  • Since this workflow requires running both Unreal Engine and a DCC application simultaneously, it can lead to crashes on less powerful computers.
  • Livelink support for some DCC applications might be limited or require custom plugins.
  • Complex scenes or high-resolution assets can result in reduced performance and frame rates.
  • Live Link may not support all custom rig features or constraints used in external software.


Best Practics

  • Set up the Live Link plugin in both the DCC application and Unreal Engine.
  • Utilize rigs that are compatible with Live Link.
  • Confirm that both applications have the same frame rate settings for precise playback.
  • Maintain organization and consistent naming conventions within the DCC application scene.
