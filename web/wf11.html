Volumetric Capture

Volumetric Capture, also referred to as volumetric video, is a technology that enables capturing Real-time spatial models of moving people and things. Volumetric Capture utilises multiple cameras at different angles to capture footage and uses computer algorithms to create volumetric images using the footage. These captures can then be imported into Unreal.

Keywords: Volumetric Capture, Volumetric Video


Pain Points

  • Volumetric captures generate files with large file sizes.
  • Playback can be challenging due to the heavy file sizes.
  • High-performance systems are required to work with volumetric captures.
  • Editing animation is complex as each frame needs to be edited to make a change.
  • Since the Volumetric capture cameras capture photogrammetry, they can record any object within the volume.
    Therefore the captures require clean-up.


Best Practics

  • Ensure the volumetric capture system is properly calibrated before capturing.
  • Employ high-quality Volumetric capture systems to achieve optimal results.
  • The cleaning process can be easier by using an animated bounding box in Houdini.
  • Ensure the volumetric capture system is lit properly to avoid capturing shadows.
